# Le_L_ChatApp

This is an app where you can send your messages online using the Socket.io platform


## Getting started
* Clone the Repo ```git clone```
* Install NODE ``` npm install```
* In the Terminal run ``` npm start```
* In the Browser go ``` localhost:3030```


## Build with/ Tools
* **HTML5**
* **SASS/CSS3**
* **JavaScript**
* **Node.js**
* **Express**
* **Vue.js**
* **Socket.io**
